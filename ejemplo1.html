Tipos de datos primitivos en JAVA

Tipo int

Es un tipo de datos de 32 bites con signo.
Su rango va entre -2 a la 31 hasta 2 a la 31, menos uno.

Tipo float

El float es un tipo de datos para almacenar numeros con coma flotante de 64 bitesde precision.
Estos numeros representan un decimal en el mundo real.
